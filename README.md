
# Redux-Toolkit-Translate-App

Bu web sitesi rapidapi.com APi'si kullanılarak yapılmış 110 farklı dilde çeviri yapan bir uygulamadır. Dillerin arasında ki değiştir tuşuna basarak mevcut dilleri değitirebilirsiniz.

# Projede Kullanılan Araçlar
- React Vite
- Axios
- React-Redux
- Reduxjs/Toolkit
- React-Select
